# Roadmap

**Status:** DRAFT
**Last updated:** 2026-06-22
**Scope:** `@codenhub/skills` package publishing preparation

## Purpose

Define the required preparation steps before publishing the `@codenhub/skills` package to the npm registry.

## Current Focus

- **Skills Review & Polish**: Conduct a thorough review of all custom AI agent skills under `skills/` to ensure high quality and correct styling.
- **Licensing & Copyright Notices**: Identify third-party or open-source skills, and add their respective license files or attribution notices to prevent licensing compliance issues.

## Planned

- **Package Registration**: Publish the `@codenhub/skills` package to npm.
- **Validation**: Verify that running `npx @codenhub/skills@latest` or `pnpm dlx @codenhub/skills@latest` works seamlessly on a clean machine.
